# LendSecureSystem

A secure lending management system with a .NET backend and frontend application.

## Project Structure

```
LendSecureSystem/
├── Backend/          # .NET 8.0 Web API
└── Frontend/         # Frontend application (to be added)
```

## Prerequisites

- .NET 8.0 SDK
- Node.js (for frontend, when added)

## Getting Started

### Backend Setup

1. Navigate to the Backend directory:
   ```bash
   cd Backend
   ```

2. Restore dependencies:
   ```bash
   dotnet restore
   ```

3. Run the application:
   ```bash
   dotnet run
   ```

4. The API will be available at:
   - HTTP: `http://localhost:5057`
   - HTTPS: `https://localhost:7222`
   - Swagger UI: `https://localhost:7222/swagger`
